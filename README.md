Creare un blog con tutte le best practices viste fino ad ora (in questo esercizio Fortify non serve)

Creare un modello Article, implementando le funzionalita' seguenti:
index, pagina per visualizzare gli articoli
create, pagina con form per creare un nuovo articolo
store, funzionalita' per il salvataggio dell'articolo nel database
show, pagina per visualizzare il dettaglio dell'articolo
edit, pagina con form con il recupero delle informazioni dell'articolo da modificare
update, funzionalita' per la modifica dell'articolo nel database
Creare un modello Tag, implementando i seguenti punti:
create, pagina con form per creare un nuovo tag
store, funzionalita' per il salvataggio del tag nel database
Implementare la Many-to-Many tra i due modelli
